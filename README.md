# React skeleton

* Webpack
* ES6
* Hot-module reload

## Setup

$> npm install

## Running

$> npm run start

http://localhost:8080

## Run postgres
$> pg_ctl -D /usr/local/var/postgres -l /usr/local/var/postgres/server.log start

## Stop postgres
$> pg_ctl -D /usr/local/var/postgres stop -s -m fast
